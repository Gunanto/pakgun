Teorema Pythagoras

Dalam segitiga siku-siku, kuadrat sisi miring (hipotenusa) sama dengan jumlah kuadrat kedua sisi lainnya.

a² + b² = c²

Perhitungan Saat Ini

a² = 9

b² = 16

a² + b² = 25

c = √25 = 5

Mencari Sisi Miring (Hipotenusa)

Langkah Penyelesaian:

  1. Kuadratkan kedua sisi tegak
  2. Jumlahkan hasil kuadrat
  3. Akar kuadratkan hasilnya

Rumus: c = √(a² + b²)

Contoh: a = 6, b = 8

c² = a² + b²

c² = 6² + 8²

c² = 36 + 64 = 100

c = √100 = 10

Kapan Digunakan?

  • Mencari diagonal persegi panjang
  • Menghitung jarak terpendek
  • Mencari panjang tangga yang bersandar

Mencari Sisi Tegak

Langkah Penyelesaian:

  1. Kuadratkan hipotenusa dan sisi yang diketahui
  2. Kurangkan: c² - a²
  3. Akar kuadratkan hasilnya

Rumus: b = √(c² - a²)

Contoh: a = 9, c = 15

b² = c² - a²

b² = 15² - 9²

b² = 225 - 81 = 144

b = √144 = 12

Aplikasi Praktis

  • Menghitung tinggi bangunan
  • Mencari jarak horizontal
  • Menentukan kedalaman sumur

Kelipatan Tripel:

Dari tripel (3, 4, 5):

  • × 2: (6, 8, 10)
  • × 3: (9, 12, 15)
  • × 4: (12, 16, 20)

Apa itu Tripel Pythagoras?

Tripel Pythagoras adalah tiga bilangan bulat positif (a, b, c) yang memenuhi persamaan a² + b² = c².

Contoh verifikasi (3, 4, 5):

3² + 4² = 9 + 16 = 25 = 5² ✓

Kegunaan Tripel Pythagoras

  • Mempercepat perhitungan
  • Membantu dalam konstruksi
  • Dasar untuk soal-soal matematika
  • Aplikasi dalam arsitektur
